# tradenote
```
A simple implementation of Nativescript-Vue.
TradeNote is an application that save a note of you buy and sell manually.

Development Environment:
Nativescript Vue
Nativescript 8.1.4
Vue @vue/cli 4.0.5
NPM 6.11.2
Android

Run:
cd tradenote
tns run android --bundle --no-hmr
```
